# LabirintGenerator

Librariile pe care doresc sa le folosesc in momentul de fata sunt: random (pentru generarea de labirinturi in sine)
Scopul este sa generez un labirint care pot sa-l reglez in lungime si latime, cate "dead end-uri" poate sa aiba si sa fie rezolvat intr-o maniera cat de cat eleganta

Nimic din ce am zis sunt decizii finale, exista posibilitatea pentru imbunatatiri sau schimbare a unor functionalitati in viitor
